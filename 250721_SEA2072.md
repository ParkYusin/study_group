# 알고리즘 스터디 문제.2072


```
t = int(input())
for test_case in range(1, t + 1) :
    data=list(map(int, input().split()))
    s = sum([x for x in data if x % 2 == 1])
    print(f'#{test_case} {s}')
```

구글링 했지만 x 와 f 가 아직도 뭔지 모르겠다.  

### 유신
print(f'#{test_case} {s}')의 결과가
# **1** 5 이렇게 출력된다면
''와 f'' 두가지 출력을 비교하고
{}의 역할을 추론해봤으면 좋겠습니다.

data=list(map(int, input().split()))
s = 0
for x in data:
    if x % 2 == 1 :
        s = s + x # s += x 같은 의미
위 코드랑 s = sum([x for x in data if x % 2 == 1])
이 코드는 같습니다
